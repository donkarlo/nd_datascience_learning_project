\title{Transformer}
\date{}

\begin{document}
    \maketitle
    \cite{vaswani-2017-attention-is-all-you-need} introduces the Transformer, a sequence transduction model based entirely on attention mechanisms, without recurrence or convolution. Its main component is self-attention, which allows the model to relate all positions in a sequence directly and in parallel. The authors also introduce multi-head attention, enabling the model to attend to different representation subspaces at the same time. The paper became foundational because it showed that attention alone can effectively model long-range dependencies in sequential data.
